  • Can It Run Doom?


On top of all the innovations of Doom, the versatility of the game stands out as one of Doom's most exploited features. The low processing power of the game (by modern standards, Doom couldn't run on weaker computers that existed then) allows the game to run on almost anything that can power it, among them:
